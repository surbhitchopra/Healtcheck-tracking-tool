Quick Table Update Test Functions

Open browser console on dashboard and use these functions:

Test Functions:

  • testOPT_NC() - Test OPT_NC table update
  • testBSNL_East() - Test BSNL East DWDM table update
  • testAllFunctions() - Check all functions are available

Manual Increment Functions:

  • incrementOPT_NC() - Increment OPT_NC runs
  • incrementBSNL_East_DWDM() - Increment BSNL East DWDM runs
  • incrementBSNL_North_DWDM() - Increment BSNL North DWDM runs
  • incrementBSNL_West_DWDM() - Increment BSNL West DWDM runs
  • incrementMoratelindo() - Increment Moratelindo PSS24 runs

Expected Behavior:

  1. Function fetches fresh data from database
  2. Updates dashboard cached data
  3. Refreshes table display
  4. Manually updates specific table cell
  5. Cell gets green highlight for 2 seconds
  6. Total runs column shows updated value
