UI INVENTORY FILES FOR TIMEDOTCOM CUSTOMER
===========================================

This directory contains UI-specific inventory configuration files for the TIMEDOTCOM customer.

Files:
- inventory_config.json: Configuration settings for inventory display in the web UI
- Future inventory templates and UI-specific files

Purpose:
- UI configuration for inventory display and management  
- Web interface settings for inventory reports
- Customer-specific inventory preferences and filters
- Links to main inventory data in Script/customer_files/

Data Source:
- Main inventory data is stored in: Script/customer_files/timedotcom/inventory_files/
- UI reads and displays this data through Django web interface

Integration:
- This UI structure mirrors the Script customer_files structure
- Inventory reports generated by scripts are displayed through this UI
- Provides web-based access to inventory information
